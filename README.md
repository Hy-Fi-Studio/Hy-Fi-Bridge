# Hy-Fi-Bridge
Hy-Fi Bridge is a duel-focused PvP minigame inspired by competitive bridging combat. Players build toward enemy goals while defending their own. Includes kit presets, scoring, queues, and ranked support. Built for sweaty PvP players and skill-based servers.
